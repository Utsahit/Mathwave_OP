# Elixir & Oak — Staff Manual

**App URL:** `https://app.elixirandoak.com`

---

## 1. Taking an Order

1. Log in with your staff credentials
2. Click **Orders** → **Create New Order**
3. Select customer (or create as walk-in)
4. Browse menu by category, click items to add
5. Adjust quantities as needed
6. Add special instructions if any
7. Click **Place Order**

The order appears in the Kitchen Display automatically.

---

## 2. Processing Payments

### Customer wants to pay:
1. Open the order
2. Click **Collect Payment**
3. Choose payment method:
   - **QR Code** — Customer scans with UPI app
   - **Payment Link** — Send link to customer's phone
   - **Cash** — Mark as paid in cash
4. Once paid → Order marked as Completed

---

## 3. Kitchen Display

The Kitchen Display shows all active orders in real-time.

### Order Statuses:
- **Pending** — New order, not yet started
- **Preparing** — Chef started working on it
- **Ready** — All items prepared, waiting for pickup

### Marking Items Ready:
1. Tap an order to expand
2. Check off each item as it's prepared
3. When all items checked → Order moves to "Ready"
4. Server receives notification to serve

---

## 4. Managing Reservations

### Walk-in Customer:
1. Click **Reservations**
2. Click **Create Reservation**
3. Enter name, phone, party size
4. Assign table
5. Click **Confirm**

### Customer Calls to Modify:
1. Find reservation by name/phone
2. Click **Edit**
3. Update date, time, party size
4. Click **Save**

---

## 5. Inventory Check

1. Click **Inventory**
2. Search for an item
3. View current stock level and reorder threshold
4. If stock is low (highlighted red), inform manager

---

## 6. Common Tasks

### Void an Item:
- If an item needs to be removed from an active order, ask manager for approval

### Apply a Discount:
- Manager approval required
- Manager selects coupon/discount from admin panel

### Mark Order as Served:
- When food reaches the table, tap **Mark Served**

---

## 7. Login Help

- Username is your email address
- Forgot password → Click "Forgot Password" on login screen
- Ask manager for account creation
